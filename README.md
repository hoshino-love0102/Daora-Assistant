# Daora Assistant

DAORA 개발 커뮤니티 서버를 위한 전용 디스코드 봇입니다.

신규 유저 환영, 역할 선택, 프로젝트 모집, 스터디 모집, 공모전/해커톤 정보 공유 등을 도와주는 서버 운영 보조 봇입니다.

## 현재 기능

- 신규 멤버 환영 메시지 전송
- 관리자 명령으로 디스코드 서버 안에서 역할 패널 설정
- 서버 역할/서버 이모지 기반 반응 역할 패널 발행
- 반응 이모지 클릭 기반 자동 역할 부여/제거

## 개발 예정 기능

- `/help` 명령어
- 프로젝트 모집 임베드 생성
- 스터디 모집 임베드 생성
- 공모전/해커톤 정보 등록
- 자기소개 안내
- 관리자 전용 공지 명령어

## 기술 스택

- Node.js
- discord.js
- dotenv

## 역할 부여 사용 방법

1. 서버에 역할과 사용할 커스텀 이모지를 먼저 준비합니다.
2. 관리자 전용 비공개 채널에서 패널 명령어로 역할 패널을 구성합니다.
3. `!패널 발행 패널이름 | #채널` 로 실제 역할 채널에 발행합니다.
4. 유저가 반응 이모지를 클릭하면 해당 역할이 추가되고, 반응을 취소하면 역할이 제거됩니다.

사용 명령어:

- `!패널 도움말`
- `!패널 생성 패널이름`
- `!패널 설명 패널이름 | 설명`
- `!패널 추가 패널이름 | @역할 또는 역할명 | 이모지`
- `!패널 제거 패널이름 | @역할 또는 역할명`
- `!패널 보기 패널이름`
- `!패널 목록`
- `!패널 발행 패널이름 | #채널`
- `!패널 삭제 패널이름`

예시:

```text
!패널 생성 언어역할
!패널 설명 언어역할 | 여러분이 주로 사용하는 언어를 골라주세요!
!패널 추가 언어역할 | @Python | <:python:123456789012345678>
!패널 추가 언어역할 | @Java | ☕
!패널 발행 언어역할 | #역할받기
```

참고:

- 이제 역할/이모지를 코드 파일에서 직접 수정할 필요가 없습니다.
- 디스코드 서버 안에서 관리자가 역할 패널을 직접 편집할 수 있습니다.
- 디스코드 자체에는 이런 패널 설정 저장소가 없어서, 봇은 내부 저장 파일로 설정을 보관합니다.
