# Daora Assistant

DAORA 개발 커뮤니티 서버를 위한 전용 디스코드 봇입니다.

신규 유저 환영, 역할 선택, 프로젝트 모집, 스터디 모집, 공모전/해커톤 정보 공유 등을 도와주는 서버 운영 보조 봇입니다.

## 현재 기능

- 신규 멤버 환영 메시지 전송

## 개발 예정 기능

- `/help` 명령어
- 역할 선택 버튼(백엔드 / 프론트엔드 / 디자이너 / 기획자 역할 지급)
- Java / Python / JavaScript / rust 등 기술 스택 역할 지급
- 프로젝트 모집 임베드 생성
- 스터디 모집 임베드 생성
- 공모전/해커톤 정보 등록
- 자기소개 안내
- 관리자 전용 공지 명령어

## 기술 스택

- Node.js
- discord.js
- dotenv
